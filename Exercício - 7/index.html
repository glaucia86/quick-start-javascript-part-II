Funções

E se você precisasse realizar uma mesma coisa, porém diversas vezes? Seria horrível ficar digitando as mesmas coisas todo o tempo e sem contar que o código irá ficar totalmente imcompreendível. As Funções nos permite associar um determinado comando com uma ação.

  • function é uma palavra reservada
  • Um nome para a função.
  • (Opcional) qualquer parâmetro necessário, se faz uso de parenteses.
  • E o bloco de código a ser executado quando a função é chamada.

Quando você chama uma função, a informação que você passa para a função (dentro do parenteses) é chamado de argumentos

Veja o exemplo da sintaxe: Function

                
                function sayHello(name) {
                  console.log("Hello " + name);
                }
                
                sayHello("Mark");
