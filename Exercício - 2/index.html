Uma outra maneira de usarmos um controle de fluxo ou escrever um determinado código de uma maneira mais elegante é justamente usando um switch case. Esta é uma maneira parecida assim como usado com o if/else. Porém, evita-se utilização de muitos 'ifs' o que deixa o código mais clean e mais elegante.

Veja o exemplo da sintaxe: Switch

switch(n) {
case examplo1:
    console.log("Este código executará se n === examplo1 for true.");
    break;
case examplo2:
    console.log("Este código executará se n === examplo2 for true.");
    break;
case examplo3:
    console.log("Este código executará se n === examplo3 for true.");
    break;
default:
    console.log("Default será executado se os cases não forem encontrados.");
    break;
}
