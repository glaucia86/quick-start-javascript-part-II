Um outro tipo de laço existente é o do...while. Executa uma vez e continua se uma determinada condição continuar true. Muito utilizado quando você não sabe quantas vezes você precisará executar o seu laço.

Veja o exemplo da sintaxe: Do...While

                //do...while loop
                var x = 1;
            
                do{ 
                   console.log("Hello!");
                   i++;
                }
                  while(x === 1);
                
                
                //while loop
                var x = 1;
                
                while(x === 2){
                   console.log("Hello!");
                 }
