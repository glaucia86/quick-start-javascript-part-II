Laços

Laços são outras formas de controle de fluxo. Eles executam o "loop" através dos blocos de código até que uma determinada condição específica não seja verdadeira.

O laço For são ótimos tipos de laços. Sempre verificam o ciclo do código para evaluar se uma determinada condição é verdadeira ou falsa. Caso o contrário, irá da um loop infinito derrubando assim o browser

Veja o exemplo da sintaxe: For

                //mostrará os números de 1-10 no console
                
                for( i = 0; i < 11; i++){
                  console.log(i);
                }
